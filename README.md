# Git Dash Dashboard

Neon-themed Git dashboard mockup built with HTML, CSS, and JavaScript. It includes profile-style sections, repo cards, activity timeline, branch health, and a commit heatmap placeholder.

## Files

- index.html
- styles.css
- main.js

## Run

Open index.html in any modern browser. No build tools required.

## Customize

- Update text, links, and images in index.html
- Adjust colors and layout in styles.css
- Tweak heatmap behavior in main.js
